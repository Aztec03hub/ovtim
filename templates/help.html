Setup Guide
Thank you for purchasing the Omron Microscan VT-Interface Module. To begin using your device, follow the instructions below:
  1. Connect a compatible barcode reader via Ethernet and power it on. If you have a second barcode reader, connect and power it on as well.
  2. Connect a keyboard and mouse to the Omron Microscan VT-Interface Module, and an HDMI cable from the device to a monitor or other display.
  3. Ensure the Omron Microscan VT-Interface Module is powered on. After the device has booted, use the Chromium web browser to navigate to http://localhost:5000/managescanners.
  4. Click on the "Add New Profile" button.
  5. Configure the bottom- and/or top-side scanner using the Automatic method (supported by some Omron smart barcode readers) or Manual method by specifying the communication settings for the device. Save your changes when finished.
  6. It's possible to create multiple profiles. This can be used, for example, when moving the Omron Microscan VT-Interface Module to another production line with a different set of barcode readers. Select the active profile on the Manage Scanners screen and click Save when finished.
  7. Connect an RS-232C serial cable from the port labeled To Inspection Machine to the Omron VT-S series inspection machine in either port labeled "RS-232C".
  8. Using the inspection machine software, from the Main Menu navigate to Machine Settings -> System Settings -> Communication Settings, select the relevant RS-232C port from the dropdown menu, and click the Set button to save your changes.
  9. Using the Omron Microscan VT-Interface Module Chromium web browser, navigate to http://localhost:5000, confirm that your scanner(s) are connected and accessible, then click "Start Barcode Monitoring".
  10. For any scanner which supports Weblink, the weblink interface will be shown on this page. Additionally, whenever a scanner successfully scans a barcode, the barcode information will be displayed. Whenever a barcode is sent to the Omron inspection machine, a message will be displayed.

About the Omron Microscan VT-Interface Module
The Omron Microscan VT-Interface Module was developed by Karl Long, an Omron AOI Application Engineer who recognized the industry need for an integrated device that could read barcode data from multiple inputs and automatically send the correct information to the AOI inspection machine. Fellow AOI Application Engineer David Young assisted with some of the software development.
The original device, unofficially and internally referred to as the Karlbox (reinforcing the stereotype that engineers are very clever when naming their creations), was powered by a Raspberry Pi and contained hardware and cabling which handled the barcode input data. The current version has incorporated several notable improvements, including handling all input data digitally (which allows on-the-fly logging, verification, and logic-based manipulation of barcode data), improved construction materials and design, and a flexible profile-oriented configuration which allows fast, simple setup for a wide range of devices and scenarios.
The VT-Interface Module is being actively improved, but feature requests or inquiries about the latest product model can be directed to the AOI support email address at AOISupport-us-a@omron.com.
